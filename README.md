# Self-Evolving Skills

`self-evolving-skills` is a proposal-first Hermes skill prototype for safely turning conversation evidence into reviewable skill evolution candidates. It does not autonomously modify memory, install skills, create cron jobs, call external APIs, or publish content.

`self-evolving-skills` 是一个 proposal-first 的 Hermes skill 原型，用于安全地把会话证据转化为可审查的 skill 演进候选项。它不会自主写 memory、安装 skill、创建 cron、调用外部 API 或发布内容。
